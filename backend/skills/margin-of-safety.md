# Margin of Safety 评估

Benjamin Graham 原话："安全边际是投资的核心概念。" 安全边际不是"便宜"，是**为误判和坏运气预留的缓冲**。

## 判断流程

1. **估内在价值**：用保守假设下的 DCF / owner earnings 估一个区间，而非点值。悲观 / 中性两个情景，别用乐观情景。
2. **算折扣**：`(内在价值下限 - 当前市值) / 内在价值下限`
3. **要求**：
   - 大盘龙头、生意稳定 → 折扣 ≥ 30%
   - 中小盘 / 周期股 / 不确定性高 → 折扣 ≥ 50%
   - 垃圾 / 困境反转 → 折扣 ≥ 70% 且有清晰的修复路径
4. **否则放弃**。不强求。宁可错过，不能买贵。

## 红旗（即便折扣够大也要退后）

- 商业模式正在被技术/监管颠覆
- 管理层有稀释股东 / 关联交易 / 财务造假嫌疑
- 不在 circle of competence 内（不懂就不买）
- 需要"假设未来三年增长 20%+"才能给出折扣

## 输出格式建议

回答用户时请给出：
- 估值区间（下限 / 上限）及关键假设
- 当前价对内在价值下限的折扣 %
- 该 case 要求的最低折扣 %
- 结论：满足 → Buy 候选；不满足 → Wait / Pass
